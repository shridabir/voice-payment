# FinCoach - Voice Financial Literacy Assistant

## Setup
1. Install: `pip install -r requirements.txt`
2. Get API keys:
   - Nessie: http://api.nessieisreal.com/
   - Anthropic: https://console.anthropic.com/
3. Update .env with your keys and account ID
4. Run: `python app.py`

## Usage
- "What's my balance?"
- "Can I afford a $200 purchase?"
- "Where does my money go?"
- "What's compound interest?"

## Features
- Real-time voice conversation
- LLM with function calling (Claude Sonnet 4)
- Real banking data from Nessie API
- Financial literacy teaching
