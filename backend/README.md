# Voice Payment Backend

## Setup
1. Get Nessie API key from http://api.nessieisreal.com/
2. Add to `.env`: `NESSIE_API_KEY=your_key`
3. Install: `pip install -r requirements.txt`
4. Update `DEMO_CUSTOMER_ID` and `DEMO_ACCOUNT_ID` in `app.py`
5. Run: `python app.py`

## Test
- Backend runs on http://localhost:5000
- Test endpoint: http://localhost:5000/api/accounts
